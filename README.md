# -renci-Ders-Se-imi-Aray-z-
Java Swing kütüphanesi kullanarak geliştirdiğim bu proje, öğrencilerin ders seçimi yapabileceği bir kullanıcı arayüzü sunar. Yazılım mühendisliği eğitimimin ilk döneminde, görsel bileşenleri düzenleme ve listeleme mantığını kavramak amacıyla tasarlanmıştır. Sade tasarımıyla temel Java masaüstü programlama pratiklerini içerir.
